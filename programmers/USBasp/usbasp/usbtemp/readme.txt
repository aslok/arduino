Программа и прошивка для чтения датчиков температуры DS18x20 через переделанный программатор USBasp:

Основана на проекте http://code.google.com/p/mikrowerk/wiki/USBTemp

Содержание архива:
firmware - прошивка для микроконтроллера USBasp.Исходный код можно найти на http://code.google.com/p/mikrowerk/wiki/USBTemp
source - исходный код программы чтения датчиков.
win - откомпилированная программа из папки source для Windows .Драйвера http://homes-smart.ru/upload/arduino/USBavr-driver.zip

Моя страничка проекта http://homes-smart.ru/index.php/oborudovanie/uroki-i-primery/38-podklyuchenie-datchikov-ds18b20-k-kompyuteru
